# hacks-ai-perm-2023

## СмотрИИшка
Решение представлено в виде набора скриптов в рамках единого блокнота Jupyter - Skyori_rutube.ipynb

Перед запуском необходимо на одном уровне с блокнотом разместить конфигурационные файлы spacy_*.cfg из данного репозитория, а также оригинальный файл с обучающим множеством хакатона
ner_data_train.csv и файл с примерами для вычисления сабмитов ner_test.csv.
Параметр DO_TRAIN отвечает за необходимость обучения моделей, если обучение уже проводилось и имеются обученные модели то они будут загружены вместо запуска процесса обучения.

В результате работы скриптов формируются следующие объекты:

Веса обученых моделей:

* base_model_bert
* spacy_multilingual
* spacy_rubert

Файлы для отправки в лидерборд на основе ner_test.csv:
* submission_base_rubert.csv
* submission_spacy_multilingual.csv
* submission_spacy_rubert.csv
, а также файл полученый при объединении разметок, полученых всеми тремя моделями: submission_hybryd.csv
